# NLP Preprocessing Engine & Chatbot

## 📌 Overview
This repository contains completed internship tasks for NLP — including:
1. Task 3: A Chatbot built using Hugging Face Transformers
2. NLP Text Preprocessing Engine
3. Sentiment Analysis examples

This project demonstrates modern NLP techniques using Python.

---

## 🤖 Task 3 — Chatbot using Hugging Face
The main notebook:
`Task3_NLP_Chatbot_HuggingFace.ipynb`

Features:
- Loads a pre-trained conversational model
- Accepts user input in an interactive loop
- Generates human-like responses

Built using the transformers library.
